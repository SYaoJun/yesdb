

![yesdb](/logo/yesdb2.png)

- A storage engine for learning

## Features

* Embeddedable library `bitcaskcpp`
* Predictable read/write performance
* Low latency
* High throughput
## How To Install
```
mkdir build 
cd build
cmake .. 
make -j `nproc`
```
